# skill-playwright

A Claude skill for complete browser automation with Playwright.

## Trigger
Use when automating browser interactions, testing web UIs, filling forms, taking screenshots, or validating web functionality.

## What it does
- Auto-detects dev servers
- Writes clean test scripts to `/tmp`
- Supports screenshots, form fills, link checks, login flows, responsive design testing

## Files
- `SKILL.md` — skill instructions
- `API_REFERENCE.md` — Playwright API reference
- `package.json` — dependencies
